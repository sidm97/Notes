HTML

  • The head element contains information about the webpage
  • the body element represents the visible content shown to the user

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element

Git

  • Process to push a branch change back to remote repository
    1. git pull origin main ensures that the latest version of the repository is in the local directory before pushing a change back into the remote repo
    2. git status: checks what branch i am currently on
    3. git checkout -b branch-name: creates a new branch and switches to it
    4. git status: checks what untracked changes have been made on new branch
    5. git add -A lists changes made to a branch waiting for commit, tracking them
    6. git commit -m "" commits changes and adds a message as a reminder as to what changes were made
    7. git push origin branch-name creates the new branch in the remote repo, allowing a pull request to be created on github

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
