Git

  • git status: checks what branch i am currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • To push a change into a remote repository
    1. git status to check what branch
    2. git checkout -b feature/xxx to create new branch to work on
    3. git status to confirm on the new branch
    4. git add -A to add the modifications made to the new branch
    5. git commit -m "message here" to commit this changes to the branch and adding a message to tag what was done
    6. git pull origin main to update the main branch on the local repo with the source in the remote repo
      7. git push origin feature/xxx to push the changes in the feature branch to the origin
    7. go on Github to create and merge pull request

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
